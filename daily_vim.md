##### Use of session
1. Save the session 
   `mksession <session_name.vim>`

2. Reload the session
   `vim -S <session_name.vim>` 

#### Display all characters
    `:set list`

#### Manning 
    `:help reg`

#### display the contents in the register 1
    `reg 1`

#### ls in vim
    * Display all the installed plugins
    `:scriptnames`
    * List all the functions used in plugins
    `:function`
